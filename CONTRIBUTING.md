## 📁 ไฟล์แปลอยู่ที่ไหน

ไฟล์แปลทั้งหมดอยู่ใน `src/BepInEx/Translation/th/Text/` จัดโครงสร้างตามต้นฉบับอังกฤษ (PriconneRe-TL)

รูปแบบ: ข้อความธรรมดา หนึ่งคู่ต่อบรรทัด
```
ญี่ปุ่น=ไทย
```

- ขึ้นบรรทัดใหม่ในเกม ใช้ `\n` (backslash + n) ไม่ใช่กด Enter จริง
- คีย์ภาษาญี่ปุ่นด้านซ้าย **ห้ามแก้** — เกมใช้คีย์นี้จับคู่คำแปล
- แก้เฉพาะค่าด้านขวา (ภาษาไทย)


## 📖 อภิธานศัพท์ (Glossary) — ใช้ให้ตรงกัน

คำศัพท์การต่อสู้/สถานะ ใช้ตามนี้เสมอ:

| ญี่ปุ่น | ไทย | ญี่ปุ่น | ไทย |
|---|---|---|---|
| 物理 | กายภาพ | 魔法 | เวทมนตร์ |
| 範囲 | วงกว้าง | 単体 | เดี่ยว |
| ダメージ | ดาเมจ | 無効 | ไร้ผล |
| 吸収 | ดูดซับ | クリティカル | คริติคอล |
| 混乱 | สับสน | 魅了 | ดึงดูด |
| スタミナ | สตามิน่า | 強化 | เสริมพลัง |
| 攻撃力 | พลังโจมตี | 防御力 | พลังป้องกัน |
| 命中 | แม่นยำ | 回避 | หลบหลีก |
| 行動速度 | ความเร็วการเคลื่อนไหว | 飛行 | ลอย |
| 継続回復 | ฟื้นฟูต่อเนื่อง | 継続ダメージ | ดาเมจต่อเนื่อง |
| 麻痺 | อัมพาต | 拘束 | กักขัง |
| HP吸収 | ดูด HP | TP上昇 | อัตราฟื้นฟู TP |

> ⚠️ บางคำยังไม่ลงตัว (รอตัดสิน) — `行動不能` (สตัน/หยุดการกระทำ), `調和` (สมดุล/Harmony) ถามใน Discord ก่อนใช้

**ชื่อฟีเจอร์เกม คงภาษาอังกฤษ** ไม่ต้องแปล: Connect, Princess Knight, Growsphere, RANK, Event Quest ฯลฯ

## ⚙️ คำแปลสกิล — ยึดรูปแบบ DB ทางการ

เมื่อแปลสกิล/regex ให้ยึดรูปแบบจาก DB เซิร์ฟไทยทางการเป็นหลัก:

| ญี่ปุ่น | ไทย |
|---|---|
| 味方単体 | เพื่อนคนเดียว |
| 味方全体 | เพื่อนทั้งหมด |
| 対象の味方 | เพื่อนที่เป็นเป้าหมาย |
| 敵単体 | ศัตรูตัวเดียว |
| 敵全体 | ศัตรูทั้งหมด |

โครงสร้าง: `เพิ่ม<stat>ให้เพื่อน<scope> {ตัวเลข}` — ใช้ `ให้` (ไม่ใช่ `ของ`) และวางตัวเลขไว้ท้าย

ตัวอย่าง: `味方単体の物理攻撃力を{0}アップ` = `เพิ่มพลังโจมตีกายภาพให้เพื่อนคนเดียว {0}`

ระดับ: 特大 = สูงมาก · 大 = สูง · 中 = ปานกลาง · 小 = เล็กน้อย

## ✅ ขอบเขตการแปล

**แปลได้ (ปลอดภัย):**
- UI, ข้อความระบบ, เมนู, ปุ่ม
- ศัพท์การต่อสู้/สถานะ (ตาม glossary)
- ชื่อ/โปรไฟล์ตัวละคร, ไอเทม, อุปกรณ์, สกิล (อ้าง DB ทางการได้ยิ่งดี)

**ระวัง/ข้าม (เสี่ยงแปลคลาดเคลื่อน):**
- เนื้อเรื่อง/บทพูดตัวละคร (Story/Event/Character/Home Screen) — แปลได้แต่ระวังบริบท
- **regex (`r:` / `sr:`)** — ห้ามแตะถ้าไม่แน่ใจ ตำแหน่ง `$1 $2` ผิด = ตัวเลขในเกมเพี้ยน

### 🔑 เอาคีย์ญี่ปุ่นมาจากไหนให้ชัวร์

อย่าพิมพ์ญี่ปุ่นขึ้นมาเอง — เสี่ยง mismatch (ตัวเต็ม/ตัวครึ่ง 全角/半角, ช่องว่าง, อักขระมองไม่เห็น, การขึ้นบรรทัด) แล้ว XUAT จับคู่ไม่ติด

พิมพ์เองมักอ่านได้เกือบทุกจุด แต่ถ้าเอา **100%** ให้ดึงคีย์จริงที่เกมส่งออกมา:

1. เปิดเกมเดินไปจุดที่อยากแปล (ปล่อยให้ XUAT เห็นข้อความนั้น)
2. เปิด `BepInEx/Translation/th/Text/_AutoGeneratedTranslations.txt` — XUAT จะ dump คีย์ญี่ปุ่นจริงไว้ที่นี่
3. copy คีย์นั้น (ฝั่งซ้ายของ `=`) ไปวางในไฟล์แปลให้ถูกที่ แล้วเติมคำแปลไทยฝั่งขวา

วิธีนี้ได้คีย์ตรงเป๊ะกับที่เกมส่ง ไม่ต้องเดา (ห้าม OCR/พิมพ์มือสำหรับข้อความยาว)

## 🔁 คีย์ซ้ำ (สำคัญ)

เกม (XUAT) จับคู่ด้วย **คีย์ ไม่ใช่บรรทัด** — โหลดทุกไฟล์เข้า dictionary ตอนเปิดเกม

- คีย์ซ้ำ **ไม่ทำให้เกมหน่วง** (lookup เป็น O(1) โหลดครั้งเดียวตอน start)
- ซ้ำได้**ถ้าค่าไทยตรงกันทุกจุด** — ไม่กระทบอะไร
- ⚠️ อันตรายคือคีย์ซ้ำ **ค่าต่างกัน** → ตัวไหนชนะขึ้นกับลำดับโหลดไฟล์ (ไม่แน่นอน)
- ถ้าแก้คำแปลของคีย์ที่ซ้ำ ต้อง**แก้ให้ตรงกันทุกจุด** ไม่งั้นกลายเป็น conflict
- ถ้าย้ายคำแปลข้ามไฟล์ ให้**ลบของเก่า** (แทนด้วยบรรทัดว่าง ไม่ลบทั้งบรรทัด เพื่อไม่ให้เลขบรรทัดขยับ)
