• โครงสร้างใหม่: ย้าย scr/ เป็น src/ ให้ตรงกับต้นฉบับอังกฤษ (ReTL)

• ลำดับความสำคัญการแปล: ไทยทางการ > ไทย fan-TL > อังกฤษ > ญี่ปุ่น
